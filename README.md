# Sanzz™ File Browser

File browser dengan tampilan mirip YouTube untuk menampilkan file media dari folder `/data/`.

## Fitur
- ✅ Tampilan grid mirip YouTube
- ✅ Preview gambar dan video
- ✅ Judul otomatis dari nama file
- ✅ Thumbnail otomatis dari file
- ✅ Sistem pencarian file
- ✅ Responsive design
- ✅ Watermark Ikhsan-Project
- ✅ Semua dalam satu file HTML
